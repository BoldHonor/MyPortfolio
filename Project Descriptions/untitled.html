Platform Game on SFML (sprite animation system)

Ori and the Blind Forest


This Project is a demo game based in platfroma. Traditionally PLatforma games always have had  horizontal platforms which made them functional with AABB collision detection which tends   to negelect rotation effects but now these games have evolved much further in games like "Ori and the blind forest","Hollow Knight" and "Stick-man fight" have given the genere of  platformas a new taste and twist followed up by 'hack and slash' platformas gaining fame  


The physics and collision as well as the animation system is self-coded the art used from 'itch.io'  the player characteris inspired from 'juggernaut of Dota 2'.
SAT collision is used for precise Collision and Normal detection. 
The animation system uses sprite strips of 1000px X 1000px  the animation is played through repetative shift of texture coordinates using mathematical functions .
