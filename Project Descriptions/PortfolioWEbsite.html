PortfolioWebsite using Three-JS and webGL

This Portfolio website was made usng ThreeJs and concepts of Webgl which is very similar to mordern Opengl . The main assets and models were self created  using Blender from 'Jelly-Virus animation' to 'scroll to play animation'  taking inspiration from various website like 'Apple airpod'
the majority of the website is self coded and used HTML , CSS, Javascript and BootsTrap. The website is created static so that it can be uploaded on github.
