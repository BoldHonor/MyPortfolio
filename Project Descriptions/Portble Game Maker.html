Portble Game Maker(Maker -OpenGL3.0 , Game -BGFX)

This Game Maker Works on Opengl but uses seprate libraries(self coded)  for object management  and physics which can be seperately used while exporting game thus it can use separate Graphics-api for running games
This solves problems coming with using Opengl and latest news  updates by 'Apple'  of  plans  to stop supporting OpenGl and work with 'Metal' insted also it makes  it easier to move to 'Vukan' .
The Maker exports a json file and can be read by  game-execution file also can be integrated during build of the game.

The game uses IMGUI on GLFW 3 on an opengl context  using the libraries this game-engine can easily be shifted to other GUIs . Engine uses standard image readers to import textures  which are easily available on   git-hub .  Materials and Textures can be seprately generated and assigned to each object.
